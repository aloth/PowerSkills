Four skill modules

Each module exposes Windows capabilities as simple JSON-in, JSON-out commands.

📧

Outlook

powerskills outlook

Full email and calendar control via Outlook COM automation.

  • Read inbox, search, filter
  • Send emails with attachments
  • Calendar events & meetings
  • Folder management
🌐

Browser

powerskills browser

Edge automation via Chrome DevTools Protocol (CDP).

  • List & switch tabs
  • Navigate, click, type
  • Extract page content
  • Take screenshots
🖥️

Desktop

powerskills desktop

Visual desktop control and window management.

  • Screenshots (full/window)
  • Window list & focus
  • Keystroke injection
  • Mouse control
⚙️

System

powerskills system

Shell commands, processes, and system information.

  • Execute shell commands
  • Process management
  • System info & health
  • File operations

Structured JSON everywhere

Every command returns a consistent envelope. No string parsing, no regex gymnastics.

▶ Request
{
  "skill": "outlook",
  "action": "inbox",
  "limit": 3,
  "unread_only": true
}
◀ Response
{
  "status": "success",
  "exit_code": 0,
  "data": {
    "messages": [
      {
        "subject": "Q1 Review",
        "from": "boss@corp.com",
        "date": "2026-03-22"
      }
    ]
  }
}
